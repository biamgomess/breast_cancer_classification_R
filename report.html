  • Understand the dataset structure and quality
  • Explore distributions and relationships

Target variable: diagnosis (M = malignant, B = benign)

7 7. Rule-based classifier (manual tree)

To mirror the hand-made approach from the coursework, we implement a simple rule-based classifier:

  • If radius_mean > 15 → Malignant
  • Else if perimeter_mean > 100 → Malignant
  • Else if area_mean > 750 → Malignant
  • Else → Benign
Code 
rule_predict <- function(radius_mean, perimeter_mean, area_mean) {
  if (radius_mean > 15) {
    return("Malignant")
  }
  if (perimeter_mean > 100) {
    return("Malignant")
  }
  if (area_mean > 750) {
    return("Malignant")
  }
  return("Benign")
}

rule_pred <- pmap_chr(
  test_3 |> select(radius_mean, perimeter_mean, area_mean),
  rule_predict
)

rule_results <- tibble(
  truth = test_3$diagnosis,
  pred  = factor(rule_pred, levels = levels(test_3$diagnosis))
)

conf_mat(rule_results, truth = truth, estimate = pred)
